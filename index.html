Informatique - dépannage - réparation ...

Un informaticien spécialisé est à votre service pour la résolution de toutes les pannes de votre ordinateur.

  • Installation de windows : Installation de système d'exploitation windows 11, windows 10, linux ubunto, linux mint, windows serveur...etc.

Déplacement - dépannage à domicile - Enlevement

Nous vous proposons de nombreuses méthodes de réparation afin de vous aider au mieux et de s'adapter à vos disponibilités (livraison, intervention à domicile ...)
